Model Insights

The charts below illustrate which behavioral signals most influence wellbeing predictions. The model uses gradient boosting algorithms to analyze behavioral patterns across multiple variables. Feature importance scores estimate how strongly each behavioral signal contributes to prediction accuracy. Across the dataset, stress variability, sleep deviation and menstrual cycle phase consistently emerge as the strongest predictors of wellbeing fluctuation. Understanding these relationships allows the system to produce interpretable predictions aligned with behavioral health research.
